# mem0 记忆组件集成 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 LangGraph 聊天 Agent 中集成 mem0 跨会话长期记忆，实现召回同步（Agent 回复前注入记忆）+ 存储异步（后台线程写入，不阻塞 SSE 响应）。

**Architecture:** 在现有图中 `init` 节点后插入 `mem0_recall_node`（同步），在 `extract_preferences` 节点后插入 `mem0_store_node`（异步后台线程）。新建 `app/memory/mem0_client.py` 封装单例客户端，`agent_node` 的 `_build_prompt_from_state` 追加 `<memories>` 段落。mem0 不可用时静默降级，不影响主链路。

**Tech Stack:** `mem0ai>=0.1.0`, LangGraph `StateGraph`, Python `threading.Thread` (daemon), `pydantic-settings`, `loguru`

```python
"""
mem0 长期记忆客户端单例
============================================================
职责：
  - 封装 mem0 MemoryClient 初始化（cloud / 自托管两种模式）
  - 单例保证整个进程只创建一个客户端实例
  - 未配置 API key 或 disabled 时返回 None，调用方需判断

使用方式：
    from app.memory.mem0_client import get_mem0_client
    client = get_mem0_client()   # None if disabled
    if client:
        results = client.search(query, user_id=user_id, limit=5)
"""
from __future__ import annotations

import threading

from loguru import logger

from app.core.config import settings

try:
    from mem0 import MemoryClient
except ImportError:  # pragma: no cover
    MemoryClient = None  # type: ignore[assignment,misc]

_mem0_client_singleton = None
_mem0_init_lock = threading.Lock()


def get_mem0_client():
    """返回 mem0 MemoryClient 单例；未启用或未配置时返回 None。"""
    global _mem0_client_singleton

    if _mem0_client_singleton is not None:
        return _mem0_client_singleton

    with _mem0_init_lock:
        if _mem0_client_singleton is not None:
            return _mem0_client_singleton

        if not settings.mem0_enabled:
            logger.debug("[mem0] disabled by config")
            return None

        if not settings.mem0_api_key:
            logger.debug("[mem0] no API key configured, skipping initialization")
            return None

        if MemoryClient is None:
            logger.warning("[mem0] mem0ai package not installed")
            return None

        try:
            if settings.mem0_base_url:
                client = MemoryClient(
                    api_key=settings.mem0_api_key,
                    host=settings.mem0_base_url,
                )
            else:
                client = MemoryClient(api_key=settings.mem0_api_key)
            _mem0_client_singleton = client
            logger.info("[mem0] MemoryClient initialized (base_url={})", settings.mem0_base_url or "cloud")
        except Exception as e:
            logger.warning("[mem0] initialization failed: {}", e)
            return None

    return _mem0_client_singleton
```

```python
# ============================================================
# mem0 记忆召回节点 —— 同步，在 Agent 回复前执行
# ------------------------------------------------------------
# 从 mem0 检索与本轮 human message 相关的跨会话历史记忆，
# 注入 AgentState.memories，供 agent_node 拼入 system prompt。
# mem0 不可用或超时时静默降级返回空列表，不阻断主链路。
# ============================================================

def mem0_recall_node(state: AgentState, config: RunnableConfig) -> dict:
    from app.memory.mem0_client import get_mem0_client

    client = get_mem0_client()
    if client is None:
        return {"memories": []}

    messages = state.get("messages") or []
    query = next(
        (m.content for m in reversed(messages) if isinstance(m, HumanMessage)),
        None,
    )
    if not query:
        return {"memories": []}

    user_id = (config.get("metadata") or {}).get("user_id", "default")
    try:
        results = client.search(query, user_id=user_id, limit=settings.mem0_top_k)
        memories = [r["memory"] for r in (results.get("results") or [])]
        logger.debug("[mem0] recalled {} memories for user={}", len(memories), user_id)
        return {"memories": memories}
    except Exception as e:
        logger.warning("[mem0] recall failed: {}", e)
        return {"memories": []}


# ============================================================
# mem0 记忆存储节点 —— 异步后台线程，不阻塞 SSE 响应
# ------------------------------------------------------------
# 在 extract_preferences_node 之后执行，提取本轮最后一条
# human + AI 消息，异步写入 mem0。失败静默忽略。
# ============================================================

def mem0_store_node(state: AgentState, config: RunnableConfig) -> dict:
    from app.memory.mem0_client import get_mem0_client
    from langchain_core.messages import AIMessage as _AIMessage

    client = get_mem0_client()
    if client is None:
        return {}

    messages = state.get("messages") or []
    user_id = (config.get("metadata") or {}).get("user_id", "default")

    last_human = next(
        (m for m in reversed(messages) if isinstance(m, HumanMessage)), None
    )
    last_ai = next(
        (m for m in reversed(messages)
         if isinstance(m, _AIMessage) and not getattr(m, "tool_calls", None)),
        None,
    )
    if not last_human or not last_ai:
        return {}

    mem0_messages = [
        {"role": "user", "content": last_human.content},
        {"role": "assistant", "content": last_ai.content},
    ]

    def _store():
        try:
            client.add(mem0_messages, user_id=user_id)
            logger.debug("[mem0] stored turn for user={}", user_id)
        except Exception as e:
            logger.warning("[mem0] store failed: {}", e)

    threading.Thread(target=_store, daemon=True).start()
    return {}


```
